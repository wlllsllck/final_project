How Its Works

Upload

Let click the Upload button to navigate to the upload page.

Click Upload File button.

Choose and upload your files which you want to upload.

The upload file will be uploaded to the server and the hash value of upload file will be stored in Blockchain.

Click Download button and check your successful upload !!

Upload

Download

Click the Download button to navigate to the download page.

Choose your file and click the link button

Generate the hash value from the download file and look up the hash value that stored in Blockchain.

Verify by checking both hash values have to be equal.

If the verification completed, Download is successful !!

Download

Verify

Click the Verify button to navigate to the verify page.

Choose your file that you want to verify

Generate the hash value from your file and then check with the hash value from the original file in Blockchain

Verify by checking both hash values have to be equal.

If the verification completed, Your file is original !!
